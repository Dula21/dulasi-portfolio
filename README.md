# 🌐 Dulasi Portfolio

A modern, responsive personal portfolio website built using **React**, **Vite**, and **Tailwind CSS**.  
This portfolio showcases my skills, education, projects, and work experience, and includes a contact form powered by **EmailJS**.

🔗 **Live Demo:** (https://dulasi-portfolio-o4x2.vercel.app/)

## ✨ Features

- ⚡ Fast and lightweight setup using **Vite**
- 🎨 Clean, modern UI with **Tailwind CSS**
- 🧩 Component-based architecture with **React**
- 🧠 Animated sections using scroll reveal effects
- 💼 Dedicated sections for:
  - About Me
  - Skills (Frontend & Backend)
  - Education
  - Work Experience
  - Projects
- 📬 Contact form integrated with **EmailJS**
- 📱 Fully responsive (mobile, tablet, desktop)
- 🌍 Deployed on **GitHub Pages**

---

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Email Service:** EmailJS
- **Deployment:** GitHub Pages

## 🚀 Getting Started (Local Setup)

### 1️ Clone the repository
```bash
git clone https://github.com/Dula21/dulasi-portfolio.git
cd dulasi-portfolio

```
### 2️⃣ Install dependencies
```bash
npm install
```
### 3️⃣ Run the development server
```bash
npm run dev

```
